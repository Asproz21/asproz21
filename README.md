## ⚙️ Sobre Mim
Engenheiro em formação impulsionado pela precisão das engrenagens e o vapor da inovação. No **IFSP**, forjo soluções full-stack que unem lógica mecânica, design retrofuturista e dados etéreos. Transformo conceitos em código funcional, com foco em automação e interfaces imersivas.

- 🎓 **Formação**: Bacharelado em Engenharia de Software - IFSP
- 🔧 **Foco Atual**: React, C# (.NET), MySQL, Autenticação JWT
- 🛠️ **Interesses**: Sistemas industriais, UI/UX dark mode, APIs RESTful
- 🚀 **Missão**: Criar aparatos digitais que ecoem a era vitoriana no mundo moderno

> _“O código é a engrenagem que move o éter; o vapor, o combustível da criação.”_

## 🔩 Projetos em Destaque
Explorando o equilíbrio entre funcionalidade e estética steampunk. Cada projeto é um mecanismo refinado para resolver problemas reais.

### 🔧 **Sistema de Registro de Funcionários (EEMS)**
- **Descrição**: Plataforma full-stack para gerenciamento de equipe com autenticação segura e CRUD otimizado.
- **Stack**: C# (.NET), Entity Framework, React, MySQL, JWT
- **Destaques**: APIs protegidas, dashboard responsivo, relatórios exportáveis.
- **[Repositório](https://github.com/Asproz21/EEMS)** | **[Demo](https://eems.luancosta.dev)**

### 🕰️ **Agenda a Vapor**
- **Descrição**: Organizador pessoal inspirado em cronógrafos mecânicos, com priorização inteligente e exportação de dados.
- **Stack**: Python, SQLite, Tkinter (para UI desktop)
- **Destaques**: Algoritmo de ranking por urgência, interface minimalista retro.
- **[Repositório](https://github.com/Asproz21/AgendaVapor)**

### 📡 **Painel Climático Etéreo**
- **Descrição**: Dashboard interativo para monitoramento meteorológico em tempo real, com visualizações gráficas.
- **Stack**: React, Chart.js, OpenWeatherMap API
- **Destaques**: Tema dark mode, responsivo para mobile/desktop, cache local.
- **[Repositório](https://github.com/Asproz21/WeatherDashboard)** | **[Live Demo](https://weather.luancosta.dev)**

> Todos os projetos incluem documentação completa, testes e licença MIT. Contribuições bem-vindas!
